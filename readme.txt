A few notes before you dive in:

Running the cluster is not necessary, since it has already been generated and is in the Softening and No_Softening Folders.
It is IMPORTANT that you download the ENTIRETY of the folder in order to access these files, otherwise, your computer will try to find a directory that doesn't exist or, worse, write files to a directory you may not want it to write to. 

Here are the following folders and what they include:
• Softening — Includes simulated time series for 100 N-body particles in an ASCII file "particle_n.txt" with softened gravity
• No_Softening — Includes simulated time series for 100 N-body particles in an ASCII file "particle_n.txt" without softened gravity
• Data — Includes initial attempts to run simulations relevant information for simulation runs
  • Simulation_Run — Includes simulation runs with gravitational softening and without gravitational softening.
    • Softening — Includes simulated time series for various N-body particles in an ASCII file "particle_n.txt" with softened gravity
    • No_Softening — Includes a first stab attempt at running the "No Softening" simulation

The home directory also includes the Jupyter Notebook "N_Body_Problem.ipynb" used to run and analyze these simulations as well as two MP4 files to display the results of the simulation.
